Proxy Indicator

Proxy Indicator is a tool utility so set/unset proxy settings in ubuntu in corporative networks where user and password are needed.

You can set/unset proxy settings in a click

Use your default proxy configuration to set proxy

This project is roughly inspired in uproxy.
